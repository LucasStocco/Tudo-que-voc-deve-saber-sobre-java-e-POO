1.Usando o método equals():

>> O método equals() compara o conteúdo de duas strings.
Ele retorna true se as strings tiverem a mesma sequência de caracteres e false caso contrário.

Exemplo:
String str1 = "olá";
String str2 = "olá";
System.out.println(str1.equals(str2));  // true

2.Usando o método equalsIgnoreCase():

>> Semelhante ao equals(), mas ignora diferenças de maiúsculas e minúsculas.
Retorna true se as strings forem iguais, ignorando a diferença de letras maiúsculas e minúsculas.

Exemplo:
String str1 = "olá";
String str2 = "OLÁ";
System.out.println(str1.equalsIgnoreCase(str2));  // true

3.Usando o método compareTo():

>> O método compareTo() compara duas strings lexicograficamente (baseado na ordem dos caracteres).
Retorna:
-> 0 se as strings forem iguais.
->Um valor negativo se a string de comparação for lexicograficamente menor.
->Um valor positivo se a string de comparação for lexicograficamente maior.

Exemplo:
String str1 = "abc";
String str2 = "abd";
System.out.println(str1.compareTo(str2));  // -1 (pois "abc" vem antes de "abd")

4.Usando o método compareToIgnoreCase():

>> Funciona da mesma forma que o compareTo(), mas ignorando as diferenças de maiúsculas e minúsculas.

Exemplo:
String str1 = "abc";
String str2 = "ABC";
System.out.println(str1.compareToIgnoreCase(str2));  // 0 (pois "abc" e "ABC" são iguais sem considerar a case)

Esses métodos são os mais comuns para comparar strings em Java, e cada um tem seu uso dependendo da necessidade (comparar exatamente, ignorando maiúsculas/minúsculas ou comparando lexicograficamente).
